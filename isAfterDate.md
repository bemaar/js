# isAfterDate

Check if a date is after another date.

## Implementation

```javascript
const isAfterDate = (dateA, dateB) => dateA > dateB;
```

## Usage

```javascript
const checkDates = isAfterDate(new Date(2020, 09, 10), new Date(2020, 09, 09));
```

### Output

```javascript
// checkDates
true;
```
